Cloud-Based Student Performance Analytics Dashboard
Overview

This project provides a cloud-based dashboard for analyzing student performance using uploaded datasets (CSV/Excel). It allows users to view academic trends, subject-wise and semester-wise performance, attendance correlations, and detection of at-risk students through interactive visualizations.

The system combines:

Streamlit → for data upload, filtering & text insights

Power BI → for visual analytics & storytelling

Cloud Deployment → for scalability & remote access
